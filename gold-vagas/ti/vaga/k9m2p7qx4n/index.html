Sobre a vaga

Buscamos pessoa desenvolvedora para construir e evoluir APIs e serviços em .NET, com foco em qualidade, segurança e entrega contínua — alinhado ao que o mercado tem exigido para times de produto e plataforma.

Requisitos principais

  • .NET 8+ e ASP.NET Core (Web API, minimal APIs quando fizer sentido)
  • C# sólido: async/await, LINQ, boas práticas de orientação a objetos
  • APIs REST bem modeladas (versionamento, status HTTP, contratos claros)
  • Entity Framework Core e/ou Dapper com SQL eficiente e migrations
  • Bancos SQL Server ou PostgreSQL (índices, transações, performance)
  • Testes automatizados: xUnit, testes de integração com WebApplicationFactory ou equivalente
  • Git (GitFlow ou trunk-based), code review e padrões de branch
  • Docker para desenvolvimento e publicação; noções de Kubernetes ou orquestração em cloud
  • CI/CD (GitHub Actions, Azure DevOps ou GitLab CI)
  • Autenticação/autorização: JWT, OAuth2/OpenID Connect
  • Mensageria ou integração assíncrona: RabbitMQ, Kafka ou barramento em cloud (ex.: Azure Service Bus)
  • Redis ou cache distribuído quando aplicável
  • Arquitetura limpa: camadas, Clean Architecture ou DDD tático (aggregates, domain services) conforme o time
  • Observabilidade: logs estruturados, métricas e OpenTelemetry ou APM (Application Insights, Datadog, etc.)
  • Experiência com cloud (Azure preferencialmente, ou AWS) — App Service, Functions, Key Vault, filas
  • Metodologias ágeis (Scrum / Kanban) e comunicação clara com produto e front-end

Diferenciais

  • gRPC, GraphQL ou contratos com Protobuf
  • Event sourcing, outbox pattern, idempotência em consumidores
  • Feature flags e entrega progressiva
  • Segurança: OWASP API Top 10, rate limiting, secrets em vault
